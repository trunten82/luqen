== Important ==

1. Replace placeholder secrets before using in production:
   - DASHBOARD_SESSION_SECRET (generate with: openssl rand -hex 32)
   - JWT RSA key pair (generate with: openssl genrsa -out private.pem 4096)

2. For multi-replica deployments, switch to PostgreSQL:
   --set compliance.dbAdapter=postgres
   --set compliance.dbUrl=postgresql://user:pass@host:5432/dbname

3. View logs:
   kubectl logs -l app.kubernetes.io/instance={{ .Release.Name }} -n {{ .Release.Namespace }} --all-containers
