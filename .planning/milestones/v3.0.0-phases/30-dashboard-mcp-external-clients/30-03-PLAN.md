**Filter manifest rationale:** `ToolMetadata.requiredPermission` is a single string (single-gate filter). For D-07's dual-permission tools (`dashboard_list_orgs`, `dashboard_get_org`, `dashboard_update_org`), the filter manifest uses `admin.org` (the lower-tier permission) so both `admin.org`-only and `admin.system`-holding callers see the tool in `tools/list`. The HANDLER then branches on `perms.has('admin.system')` to decide scope — system-wide vs caller's own org. Because admin role users get ALL permissions via `resolveEffectivePermissions` (permissions.ts line 90), `admin.system` holders via admin role ALSO have `admin.org` — so `admin.org` is a safe lower bound.

**Handler branching contract** for `dashboard_list_orgs`, `dashboard_get_org`, `dashboard_update_org`:
- `ctx.permissions.has('admin.system')` → the caller may target any org
- else (caller has only `admin.org`) → the caller may only target their own `ctx.orgId`; throw 403-shaped error envelope if `args.targetOrgId !== ctx.orgId`
- `dashboard_list_orgs` special case: if only `admin.org`, return `[await orgRepo.getOrg(ctx.orgId)]`; if `admin.system`, return `listOrgs()`
- `dashboard_create_org` is EXEMPT — the filter manifest pins `admin.system` directly (no branching needed).

    function parseSseOrJson(body: string): Record<string, unknown> {
      const trimmed = body.trim();
      if (trimmed.startsWith('{')) return JSON.parse(trimmed) as Record<string, unknown>;
      const dataLine = trimmed.split('\n').map((l) => l.trim()).find((l) => l.startsWith('data:'));
      if (dataLine == null) throw new Error(`No SSE data line in body: ${body}`);
      return JSON.parse(dataLine.slice('data:'.length).trim()) as Record<string, unknown>;
    }

    async function callTool(app: FastifyInstance, name: string, args: Record<string, unknown>): Promise<Record<string, unknown>> {
      const resp = await app.inject({

        headers: { authorization: 'Bearer valid-jwt', 'content-type': 'application/json', accept: 'application/json, text/event-stream' },
        payload: rpc('tools/call', { name, arguments: args }),
      });
      return parseSseOrJson(resp.body);
    }

    function extractText(parsed: Record<string, unknown>): Record<string, unknown> {
      const r = parsed['result'] as { content?: Array<{ text?: string }> } | undefined;
      return JSON.parse(r?.content?.[0]?.text ?? '{}') as Record<string, unknown>;
    }
