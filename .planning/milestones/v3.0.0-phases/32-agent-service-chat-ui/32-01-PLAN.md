Add token-level streaming support to the existing OpenAI and Ollama provider adapters and introduce a brand-new Anthropic provider adapter. This is the Wave-1 prerequisite for the agent-conversation capability (Plan 02) and ultimately AgentService (Plan 04). No existing capability is allowed to regress.

Purpose: AGENT-02 requires all LLM calls for the agent to route through the capability engine. That means the engine's provider layer must support streaming, provider-native tool calls, and a third provider (Anthropic) that has the strongest tool-use parity with MCP-shaped schemas. Without this plan, nothing downstream in Phase 32 can stream tokens to the browser.
