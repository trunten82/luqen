## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| authenticated MCP caller → tools/list | user may have pre-31.2 scope bundle; must not leak tool names beyond RBAC |
| authenticated MCP caller → tools/call | tool dispatch is the privileged operation; filter drift can allow unauthorized invocation without this guard |
| S2S caller (permissions.size === 0) → tools/list | 30.1 invariant preserved: scope-filter is authoritative; RBAC-filter would return [] (wrong) |
| new contributor → DASHBOARD_TOOL_METADATA | silent drift (add tool, forget requiredPermission) exposes tool to every authenticated user; test 3 blocks CI |
