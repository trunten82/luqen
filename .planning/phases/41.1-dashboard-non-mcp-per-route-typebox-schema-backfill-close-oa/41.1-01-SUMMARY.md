# Phase 41.1 Plan 01: OAuth surface TypeBox schema backfill — Summary

Mechanical backfill of per-route TypeBox `schema:` blocks across the 7 OAuth
route files in `packages/dashboard/src/routes/oauth/`. Six files received
schemas; `oauth/index.ts` re-exports only and was correctly left alone.

## What was delivered

### Task 1 — metadata endpoints (commit `5930b2e`)

- **well-known.ts** — `GET /.well-known/oauth-authorization-server` now declares
  `AuthorizationServerMetadataSchema` (RFC 8414) on 200, ErrorEnvelope on 4xx/5xx.
- **protected-resource.ts** — `GET /.well-known/oauth-protected-resource` now
  declares `ProtectedResourceMetadataSchema` (RFC 9728) on 200, ErrorEnvelope on
  4xx/5xx.
- **jwks.ts** — both `GET /oauth/jwks.json` and `GET /.well-known/jwks.json`
  share a common `jwksRouteOptions` block with `JwksResponseSchema` (RFC 7517)
  on 200 + ErrorEnvelope on 4xx/5xx. Per-key shape is loose
  (`additionalProperties:true`) so future JWK types are still accepted.

### Task 2 — core flow endpoints (commit `ab30b8e`)

- **authorize.ts** —
  - `GET /oauth/authorize`: `AuthorizeQuerySchema` (RFC 6749 §4.1.1 + PKCE)
    on querystring; response declares `200:Type.String()` (consent HTML),
    `302:Type.Null()` (login redirect or fast-path consent redirect), plus
    `400/401/403:ErrorEnvelope`.
  - `POST /oauth/authorize/consent`: `ConsentBodySchema` on body; response
    declares `302:Type.Null()` (allow / deny redirect to client) plus
    `400/401/403:ErrorEnvelope`.
- **token.ts** — `POST /oauth/token`: `TokenBodySchema` on body
  (grant_type / code / refresh_token / etc., form-encoded); response declares
  `200:TokenResponseSchema` (RFC 6749 §5.1 access-token success) plus
  `400/401:ErrorEnvelope`.
- **register.ts** — `POST /oauth/register`: `ClientRegistrationRequestSchema`
  on body (RFC 7591 §2 client_metadata); response declares
  `201:ClientRegistrationResponseSchema` (RFC 7591 §3.2.1) plus
  `400/401/429:ErrorEnvelope`. The 429 is reachable via the per-route
  rate-limit plugin (10 req/hr/IP) declared above.
