## Patterns Used

All admin routes follow one of these shapes:

1. **Full HTML page (admin/* page route)** — `schema: HtmlPageSchema` (or merged with `querystring`/`params`).
2. **HTMX partial (mutation/list-fragment)** — local `HtmlPartialResponse` constant with `produces: ['text/html']` + `response: { 200: Type.String(), 4xx: ErrorEnvelope }`.
3. **JSON-only handlers** — typed body + `response: { 200: <local shape>, 4xx: ErrorEnvelope }` (e.g., `proposals.ts /bulk-action`).
4. **Mixed HTML+JSON** — `system-brand-guidelines.ts` uses `Type.Union([Type.String(), Type.Object({}, { additionalProperties: true })])` because handlers branch on `hx-request` header.

All bodies/queries declared with `{ additionalProperties: true }` per Phase 41 D-05 to avoid breaking pre-existing callers that send superset fields (CSRF tokens etc.).
