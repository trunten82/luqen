## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| admin browser → /admin/digest-schedules | Authenticated admin (admin.system) manages schedules; CSRF on mutations |
| admin browser → /admin/digest-schedules/:id/view + :id/pdf | Auth-gated digest preview/download; org-scoped to the schedule's org |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-82-14 | Elevation of privilege | non-admin creating/sending/viewing digests | mitigate | every admin route preHandler: requirePermission('admin.system'); digest.manage seeded in Plan 01; rbac snapshot regenerated in Plan 05 |
| T-82-16 | Spoofing/CSRF | forged POST/PATCH/DELETE on schedules | mitigate | _csrf hidden field + hx-include="[name='_csrf']" on all mutating buttons (existing CSRF interceptor) |
| T-82-17 | Tampering (XSS) | unescaped schedule name / site URL in the TR/view | mitigate | escapeHtml on all interpolated values in buildDigestScheduleRow + {{t}}/triple-vs-double brace discipline in hbs |
| T-82-19 | Repudiation | digest UI asserting compliance/fault | mitigate | D-12 forbidden-words grep on all new hbs + json; band badge label-not-number; disclaimer card always present |
| T-82-SC | Tampering | npm installs | mitigate | No new dependencies — reuses fastify, typebox, handlebars, existing notify/pdf code |
</threat_model>
