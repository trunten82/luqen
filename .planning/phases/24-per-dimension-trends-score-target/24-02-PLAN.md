    - GET /brand-overview returns targetScore (number|null), targetY (number|null), gapValue (number|null), gapClass (string) in view data.
    - When targetScore is set: targetY is computed via computeTargetY using composite values array. gapValue = currentAvgScore - targetScore. gapClass = 'text--success' if gap >= 0, 'text--warning' if gap >= -10, 'text--error' if gap < -10.
    - When targetScore is null: targetY is null, gapValue is null, gapClass is empty.
    - POST without branding.manage permission returns 403.

1. **brand-overview.ts** — Add POST route and extend GET:

   **POST /brand-overview/target**:
   - `preHandler: requirePermission('branding.manage')`
   - Parse body: `const { target } = request.body as { target?: string }`.
   - If target is empty string or undefined, call `storage.organizations.setBrandScoreTarget(effectiveOrgId, null)`.
   - Otherwise parse to integer, validate 0-100, call `storage.organizations.setBrandScoreTarget(effectiveOrgId, parsed)`. On validation error, return 400 with error message.
   - Redirect to `/brand-overview` (or `/brand-overview?org=X` if global admin).

   **GET /brand-overview** extension:
   - After resolving effectiveOrgId, call `const targetScore = await storage.organizations.getBrandScoreTarget(effectiveOrgId)`.
   - Compute targetY: if targetScore is not null and activeSite exists and activeSite.compositeValues.length >= 2, call `computeTargetY(targetScore, activeSite.compositeValues)`. If result is -1, set targetY to null. Otherwise round to 1 decimal.
   - Compute gap: if targetScore is not null and summary.avgScore is not null, `gapValue = summary.avgScore - targetScore`. Determine gapClass: `gapValue >= 0 ? 'text--success' : gapValue >= -10 ? 'text--warning' : 'text--error'`. If either is null, gapValue = null, gapClass = ''.
   - Add to viewData: `targetScore`, `targetY`, `gapValue`, `gapClass`.
   - Also pass `canManageBranding: boolean` based on whether request user has `branding.manage` permission (check `(request as any).permissions?.has('branding.manage')`). This controls whether the target input form renders.
