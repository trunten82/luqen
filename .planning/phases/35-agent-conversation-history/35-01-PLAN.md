## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-35-01 | T (Tampering) | searchForUser SQL | mitigate | `%`, `_`, `\` escaped with explicit `ESCAPE '\\'` clause before binding; query always bound via `@q` parameter, never concatenated |

| T-35-03 | R (Repudiation) | softDeleteConversation | accept | Audit logging is wired at the route layer in Plan 03 — repo stays side-effect-free for testability |
| T-35-04 | D (DoS) | searchForUser | mitigate | `limit` capped at 50; snippet extraction bounded by `getFullHistory` slice — no unbounded scans |
