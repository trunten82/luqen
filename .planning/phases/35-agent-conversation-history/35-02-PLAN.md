    Prompt shape (internal `buildTitlePrompt`): instruct the model to return ONLY a 3–5 word
    title with no quotes, no punctuation at the end, no "Title:" prefix. Example literal:

    ```
    Summarise this exchange in 3 to 5 words. Return ONLY the title — no quotes, no trailing
    punctuation, no preface.

    User: {userMessage}
    Assistant: {assistantReply}
    ```

    `sanitiseTitle` MUST (implementation order):
    1. Trim
    2. Strip leading `Title:`, `Subject:` (case-insensitive, one occurrence)
    3. Strip surrounding quotes `"` `'` `“` `”`
    4. Strip trailing `.`, `!`, `?`
    5. Collapse internal whitespace
    6. Truncate to 80 chars hard ceiling (defence in depth; model is asked for ≤5 words)
    7. Return `''` if empty after cleanup — caller substitutes fallback

    `fallbackTitle`: `userMessage.replace(/\s+/g, ' ').trim().slice(0, 50)`.

    The outer `generateConversationTitle` call MUST wrap the LLM call in try/catch and on ANY
    throw return `fallbackTitle(args.userMessage)` without retry. On success, return
    `sanitiseTitle(accumulatedText) || fallbackTitle(args.userMessage)`. Pass `tools: []`,
    `contextHintsBlock: ''`, `onFrame: () => {}`, and forward `args.signal`.

    Follow @CLAUDE.md: small file (<200 lines), explicit error handling (no silent swallow at
    library boundary — we log nothing here because caller fires-and-forgets; note in JSDoc),
