## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Server JSON response → DOM | Response payload is trusted (comes from same-origin org-scoped route) but snippet content is originally user-typed message — still must be escaped |
| Client form input → network | CSRF token required; token injected from `<meta>` tag |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-35-16 | T (XSS) | renderSnippetWithMark | mitigate | Zero innerHTML; `<mark>` built via createElement + textContent; snippet prefix/suffix via createTextNode — scripts cannot execute even if snippet contains HTML |
| T-35-17 | T (CSRF) | POST rename / POST delete | mitigate | `x-csrf-token` header on every POST from `agent.js` pulling from `<meta name="csrf-token">`; @fastify/csrf-protection rejects missing/invalid |
| T-35-18 | I (Info disclosure) | search autocomplete | mitigate | Debounced 250 ms minimises network fan-out; no query echoed to a global log; results list lives within org-scoped `#agent-history-list` only |
| T-35-19 | D (DoS) | IntersectionObserver loop | mitigate | `isFetchingMore` guard + `nextOffset === null` termination + hard cap at `limit=50` per page (server enforced) |
</threat_model>
