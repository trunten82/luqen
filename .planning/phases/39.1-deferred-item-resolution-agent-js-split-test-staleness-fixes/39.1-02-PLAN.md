window.__luqenAgent (one-way export from agent.js):
```
window.__luqenAgent = Object.freeze({
  // DOM helpers
  byId: (id: string) => HTMLElement | null,
  announce: (msg: string) => void,                       // aria-live announcements

  // CSRF + identity
  csrfToken: () => string,
  getConversationId: () => string | null,
  setConversationId: (id: string | null) => void,

  // i18n bridge
  formatToolI18n: (key: string, vars?: Record<string,string>) => string,
  readToolI18n: () => Record<string, string>,            // reads JSON block from agent-drawer.hbs

  // Markdown + clipboard
  renderMarkdownInto: (el: HTMLElement, markdown: string) => void,
  recordMarkdownSource: (el: HTMLElement, src: string) => void,
  readMarkdownSource: (el: HTMLElement) => string | null,
  writeToClipboard: (text: string) => Promise<void>,     // writeText fallback for JSDOM
});
```

Module load order in agent-drawer.hbs (defer, in this order):
  1. agent.js          — defines window.__luqenAgent + drawer toggle + openStream + ensureAssistantBubble + init()
  2. agent-history.js  — IIFE, reads __luqenAgent
  3. agent-tools.js    — IIFE, reads __luqenAgent
  4. agent-actions.js  — IIFE, reads __luqenAgent
  5. agent-org.js      — IIFE, reads __luqenAgent

IIFE skeleton each module follows:
```js
;(function () {
  'use strict';
  const A = window.__luqenAgent;
  if (!A) return; // agent.js failed to load — fail closed
  // ... module-scoped state + document-level listeners
})();
```
