<task type="auto">
  <name>Task 1: Provision a clean container and run install.sh end-to-end</name>
  <read_first>
    - install.sh (current state — should be the patched version from Plan 40-03)
    - .planning/phases/40-documentation-sweep/40-03-DELTA.md (env var list — needed for the run command)
  </read_first>
  <action>
    Provision a clean container. Two acceptable paths:
    - **LXC**: `lxc launch ubuntu:22.04 luqen-dryrun-$(date +%s)` then `lxc exec <name> -- bash`
    - **Docker**: `docker run --rm -it -v $(pwd):/luqen -w /luqen ubuntu:22.04 bash` (install curl/git/node prerequisites first)

    Inside the container:
    1. Clone or bind-mount the repo at the v3.1.0 head commit
    2. Export every required env var from the DELTA file using safe development defaults
    3. Run `bash install.sh 2>&1 | tee /tmp/install-transcript.log`
    4. Capture exit code

    Copy `/tmp/install-transcript.log` out of the container into `.planning/phases/40-documentation-sweep/40-07-DRYRUN.md` under section `## Install transcript` as a fenced code block.

    If install.sh exits non-zero: stop here, capture the failure log section, and proceed to Task 4 (gap feedback).
  </action>
  <acceptance_criteria>
    - `test -f .planning/phases/40-documentation-sweep/40-07-DRYRUN.md` → 0
    - File contains `## Install transcript` section
    - File contains the literal line `Install exit code: 0` OR proceeds to Task 4 with the failure documented
  </acceptance_criteria>
</task>
