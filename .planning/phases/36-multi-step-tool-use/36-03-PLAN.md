<objective>
Rewire AgentService to use parallel dispatch, capture rationale, manage a shared per-turn retry budget, and emit per-tool stream events. This is the single biggest behaviour change of the phase — backed by the dispatcher + SSE frame extensions from 36-02 and the audit-row schema from 36-01.

Purpose: Implements ATOOL-01, ATOOL-02, ATOOL-03, ATOOL-04 in the agent loop.

Output:
- `runTurn` dispatches a tool_use batch via `dispatchAll`, in original order, results persisted in original order, audit rows written with rationale + outcome.
- New helper `extractRationale(turn)` gathers `turn.text` (any provider) plus optional `turn.thinking` (Anthropic) into a single string, returning null if empty.
- `AgentStreamTurn` interface extended with optional `thinking?: string` so providers that surface thinking blocks can pass them through.
- New helper `buildRetryGuidance(failureSentinel, budgetRemaining)` produces the string appended to a failed tool_result content.
- Shared `retriesRemaining` counter starts at 3 per `runTurn`; each non-success outcome decrements; when 0, guidance string is omitted from subsequent failures.
- Per-tool SSE frames emitted: `tool_started` immediately before `dispatchAll` (one frame per call in input order); `tool_completed` as each individual promise settles (status: success | error, plus errorMessage when error).
</objective>

       ```ts
       export function extractRationale(
         turn: { readonly text: string; readonly thinking?: string },
       ): string | null {
         const parts: string[] = [];
         const t = (turn.thinking ?? '').trim();
         if (t.length > 0) parts.push(t);
         const x = turn.text.trim();
         if (x.length > 0) parts.push(x);
         return parts.length === 0 ? null : parts.join('\n\n');
       }

       export function buildRetryGuidance(
         result: ToolDispatchResult,
         retriesRemaining: number,
       ): string | null {
         // Success path: anything without an `error` sentinel.
         if (typeof result !== 'object' || result === null || !('error' in result)) {
           return null;
         }
         const sentinel = (result as { error: string }).error;
         if (retriesRemaining > 0) {
           return `Tool call failed: ${sentinel}. You have ${retriesRemaining} retry attempt(s) remaining for this turn — you may retry with different arguments, switch to another tool, or proceed with what you have.`;
         }
         return `Tool call failed: ${sentinel}. The retry budget for this turn is exhausted; do not retry this tool. Proceed with the information you have.`;
       }
       ```
