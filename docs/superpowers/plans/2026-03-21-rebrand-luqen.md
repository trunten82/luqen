# Rebrand to Luqen — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rename all "pally" / "pally-agent" / "Pally" references to "luqen" / "Luqen" across the entire monorepo, releasing as v1.0.0.

**Architecture:** Scripted search-and-replace in ordered passes, with manual verification between passes. Each task targets one layer of the codebase. The approach is: replace strings in file contents first, then rename files/directories, then rebuild and verify.

**Tech Stack:** bash (sed, find), npm, TypeScript build toolchain

**Spec:** `docs/superpowers/specs/2026-03-21-rebrand-luqen-design.md`

---

## Important Notes

- **pa11y references stay unchanged** — pa11y is the upstream dependency, not our brand. Only replace "pally" (without the number), never "pa11y".
- **CHANGELOG.md historical entries stay unchanged** — they document what happened at that version.
- The rebrand design spec itself (`docs/superpowers/specs/2026-03-21-rebrand-luqen-design.md`) documents both old and new names by design — do NOT blindly replace inside it. Same for this plan file.
- **package-lock.json** will be regenerated by `npm install` — do not manually edit it.
- **dist/ directories** will be regenerated by build — do not manually edit them.

### Task 9: GitHub Repository Rename (Manual)

This must be done last, after the push.

- [ ] **Step 1: Rename the GitHub repository**

Go to https://github.com/trunten82/pally-agent → Settings → Repository name → Change to `luqen`.

Or via CLI:

```bash
gh repo rename luqen
```

- [ ] **Step 2: Update the local git remote**

```bash
git remote set-url origin https://github.com/trunten82/luqen.git
```

- [ ] **Step 3: Create the GitHub release**

```bash
gh release create v1.0.0 --title "v1.0.0 — Luqen" --notes "$(cat <<'EOF'
## Rebrand: pally-agent → Luqen

Complete rebrand to **Luqen**.

### Breaking Changes
- All packages renamed from `@pally-agent/*` to `@luqen/*`
- CLI: `pally-agent` → `luqen`, `pally-dashboard` → `luqen-dashboard`, etc.
- MCP tools: `pally_scan` → `luqen_scan`, etc.
- Env vars: `PALLY_*` → `LUQEN_*`
- Config: `.pally-agent.json` → `.luqen.json`

All features from v0.22.0 are unchanged. This is a naming-only change.
EOF
)"
```
